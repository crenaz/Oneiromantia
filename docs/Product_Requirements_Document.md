## Óneiro 🌙 — Agentic Dream Journal & Pattern Analyst

### Next.js 15 + TypeScript + Tailwind CSS + shadcn/ui

---

# Product Vision

Óneiro is an AI-powered dream intelligence platform that transforms unstructured dream journals into an interactive, visually immersive, and analytically rich experience.

The application leverages a Python-based multi-agent system that ingests dream logs, extracts recurring symbols, identifies Jungian archetypes and emotional trajectories, and generates procedural p5.js artwork representing the subconscious landscape of the user.

The frontend should feel less like a traditional dashboard and more like a digital observatory of the subconscious—combining AI explainability, creative visualization, and long-term personal insight.

# p5.js Visualization Integration

The Python agentic backend generates p5.js sketches that are rendered directly in the UI.

Requirements:

* Embed sketches within dedicated viewer components using `<iframe>` or a secure sandboxed rendering strategy.
* Support responsive resizing while preserving aspect ratio.
* Allow full-screen viewing and playback controls where applicable.
* Enable regeneration and version history for multiple visualizations of the same dream.
* Lazy-load sketches for performance.
* Pause off-screen animations to reduce CPU/GPU usage.
* Provide metadata overlays (generation date, associated dream, symbols, archetypes).
* Support downloadable exports (PNG, SVG if available, original p5.js bundle).

Artwork should appear throughout the application, including the Dashboard, Dream Gallery, Dream Detail pages, Symbol Library, Archetype Explorer, and AI Insights, creating a cohesive visual language across the platform.

# Accessibility & Performance

* Meet WCAG 2.2 AA accessibility standards.
* Ensure full keyboard navigation and visible focus states.
* Respect reduced-motion user preferences while preserving essential feedback.
* Lazy-load heavy visualizations and route-level code splits.
* Optimize p5.js asset loading and suspend inactive sketches.
* Target Core Web Vitals with responsive rendering and efficient caching.

---

# Overall Experience

Óneiro should present itself as an **AI-powered observatory of the subconscious**—a place where reflective journaling, explainable multi-agent intelligence, and generative art converge. The interface should balance immersive storytelling with transparent analytics, allowing users to seamlessly move from writing a dream to exploring symbol networks, emotional trajectories, and procedurally generated visualizations. Every interaction should reinforce the feeling of exploring a living, evolving map of the user's inner world, making the application both emotionally engaging and technically trustworthy.
